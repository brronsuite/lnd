lnwire
======

[![MIT licensed](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/brronsuite/lnd/blob/master/LICENSE)
[![GoDoc](https://img.shields.io/badge/godoc-reference-blue.svg)](http://godoc.org/github.com/brronsuite/lnd/lnwire)

The lnwire package implements the Lightning Network wire protocol.

This package has intentionally been designed so it can be used as a standalone
package for any projects needing to interface with lightning peers at the wire
protocol level.
